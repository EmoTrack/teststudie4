Zur Installation mit Firefox oder Google Chrome:
Klicken Sie rechts auf die drei kleinen Punkte um die Browser-Optionen zu öffnen.
Dann können Sie die Option Installieren auswählen
Danach können Sie dieses Fenster schließen und die App über das Icon starten


Zur Installation mit dem Samsung-Browser:
Klicken Sie rechts auf die drei Linien um die Browser-Optionen zu öffnen.
Dann können Sie die Option Seite hinzufügen zu auswählen und dann auf zum Startbildschirm klicken.
Danach können Sie dieses Fenster schließen und die App über das Icon starten
